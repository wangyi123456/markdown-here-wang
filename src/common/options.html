MD快速排版

少即是多，三套严选专业设计模板，涵盖全部排版场景，并且支持自定义样式，帮你快速排版

📖 快速上手
1
在任意富文本编辑器中，用 Markdown 语法写内容
2
点击工具栏的 按钮或按 Alt+Shift+M
3
内容即刻渲染为 HTML，再次点击可撤销
⚠️ 快捷键冲突！修改
🎨 渲染风格
选择主题后，工具栏按钮和右键菜单均使用此风格渲染。也可在编辑区右键 →「切换渲染风格」快速切换。

🖌️ 微调样式
基于你当前选择的渲染风格模板，你可以直接在此通过修改 CSS 来微调排版效果。更改将自动保存并应用于下次渲染。
